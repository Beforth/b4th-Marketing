# Marketing Module - Standalone

This is a complete marketing module with the same beautiful design as the HR system.

## Features
- Campaign Management
- Lead Management  
- Email Templates
- Analytics & Metrics
- Lead Scoring
- Activity Tracking

## Installation
1. Copy this folder to your project
2. Add 'marketing_standalone' to INSTALLED_APPS
3. Run migrations
4. Add URLs to your main urls.py

## Design
Uses the same design system as HR module:
- Tailwind CSS
- Lucide Icons
- Responsive design
- Beautiful animations
